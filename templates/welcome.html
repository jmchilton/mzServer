Example Application

 Using this API it is possible to implement scalable complex mass-informatic applications and dataviewers. Chromatoplot is an example demonstrating a viewer for a large scale 3D fractionation scheme, the viewer provides instant and concurrent extraction of chromatograms from any 9 of 42 possible files entirely from within the browser!

Set up your own mzServer

 You can run your own server by downloading this zipped directory structure and running the script called mzServer.mz -- for which you will need a recent version of multiplierz and Thermo's XDK (or if you would prefer, the free MSFileReader system). If you would like to serve up WIFF files, you will need to get a copy of ProteinPilot (you do not need an actual license, since by default it serves as a free results viewer).
